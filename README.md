# laravelAuth 

### From this project I learned
- how to use user registration
- how to use access control
- how to use route garde 
- how to use CRUD
- user dashboard

## Setup

1. composer install
1. npm install
1. cp .env.example .env
1. add your database to .env
1. php artisan key:generate
1. php artisan migrate
